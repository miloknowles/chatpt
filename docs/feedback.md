# Feedback From Testing

Running list of UI and product updates noticed while testing the app.

## Chat

- Make chat bubbles more rounded.
- Reduce the vertical height of the top chat header area. This is the area with the chat selector dropdown and the button to start a new chat.
- Make the chat icon in the top right more prominent in size and color.
- Show actual conversations in the sidebar `Chat` area.
- Make the sidebar `Chat` area expandable.

## Sidebar

- Remove `Qualities` from the left sidebar for now.
- Remove `Insights` from the left sidebar for now.
- Reduce the vertical space used by the top area of the sidebar layout.

## Training

- Remove `Sessions` from the `Training` area for now.
- Keep `Sessions` in the `Program` area.
- In the session builder, make the exercise selection area take up the full vertical height. Right now padding prevents the area, including the right divider, from extending the full height.
- Add `Session Builder` at the top of the exercise selection toolbar.
